Authentication

Go2 uses API keys to authenticate requests. You can view and manage your API keys in the Go2 Dashboard under Developer Settings.

Your API keys carry many privileges, so be sure to keep them secure! Do not share your secret API keys in publicly accessible areas such as GitHub, client-side code, and so forth.

Authorization: Bearer YOUR_API_KEY

Get Team Productivity

Retrieves high-level productivity scores and time-saved metrics for a specific team. Go2's AI calculates this based on deep work blocks versus context switching.

GET https://api.go2.com/v1/teams/{team_id}/productivity

Path Parameters

NameDescription
team_id string The unique identifier of the team.

Query Parameters

NameDescription
timeframe string Optional. 7d, 30d, or 90d. Default is 7d.

Generate Coaching Suggestions

Returns personalized, AI-generated coaching recommendations for a user to help them optimize their workday, reduce burnout, and increase focus time.

GET https://api.go2.com/v1/users/{user_id}/coaching

Path Parameters

NameDescription
user_id string The unique identifier of the employee.

Create Webhook Subscription

Subscribe to real-time events, such as when a new coaching insight is generated or a burnout risk threshold is crossed.

POST https://api.go2.com/v1/webhooks

Body Parameters

NameDescription
url string The HTTPS URL where events will be sent.
events array List of event types (e.g., insight.created, burnout.alert).

200 OK
