Privacy by Design

Your team won't hate this one.

Go2 captures everything locally but only shares patterns. Surveillance is architecturally impossible at the manager level. This isn't a policy — it's how the code works.

🔒 Local-first processing

Raw keystrokes, audio, and screen data are processed on the worker's own machine. Only anonymized patterns and aggregated insights leave the device. Managers never see raw captures because raw captures never leave.

👥 Team-level only for managers

The manager dashboard shows patterns across roles and teams. There is no button to drill into an individual's keystrokes or screen. It doesn't exist. Not hidden behind a setting — absent from the code.

⏻ Workers control their coach

Each person can pause coaching, dismiss suggestions, or mark topics as irrelevant. The AI adapts. This isn't big brother — it's a personal trainer they can talk back to.

👁 Full transparency

Workers can see exactly what Go2 knows about them at any time. Complete visibility into captured patterns. No hidden logs. No secret reports to management. Trust is built on openness, not promises.
