API Usage Examples

cURL Example: Fetch Workflow Data

curl -X GET "https://api.go2.io/v1/workflow/data" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

Webhook Payload Example

{
  "event": "coaching_tip_generated",
  "user_id": "12345",
  "tip": "Optimize your email workflow by using filters.",
  "timestamp": "2023-10-01T12:00:00Z"
}

cURL Example: Post Integration Config

curl -X POST "https://api.go2.io/v1/integrations/sso" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "Okta",
    "client_id": "your_client_id",
    "client_secret": "your_client_secret"
  }'

Webhook Response Handler (Node.js)

const express = require('express');
const app = express();

app.post('/webhook', (req, res) => {
  const payload = req.body;
  console.log('Received webhook:', payload);
  res.status(200).send('OK');
});

app.listen(3000);

cURL Example: Query User Data Path

curl -X POST "https://api.go2.io/v1/path/query" \
  -H "Authorization: Bearer USER_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Show my time spent on tasks last week"
  }'
