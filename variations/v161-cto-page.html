// api
RESTful API with full CRUD
Everything in the Go2 dashboard is available via API. Build custom integrations, export data, trigger automations programmatically.
Quick Start bash
# Authenticate and fetch team insights curl -X GET https://api.go2.io/v2/teams/insights \ -H "Authorization: Bearer $GO2_API_KEY" \ -H "Content-Type: application/json" # Response { "team_id": "tm_8x7k2m", "period": "2026-03-10/2026-03-17", "productivity_gain": 0.23, "automations_triggered": 1847, "docs_generated": 34, "coaching_sessions": 892, "avg_satisfaction": 0.89 }
Webhook Configuration javascript
// Register a webhook for automation events const webhook = await go2.webhooks.create({ url: "https://your-app.com/go2/events", events: [ "automation.suggested", "automation.deployed", "doc.generated", "insight.critical" ], secret: process.env.WEBHOOK_SECRET }); // Webhook payload example { "event": "automation.suggested", "timestamp": "2026-03-17T14:23:00Z", "data": { "automation_id": "auto_9f3x", "description": "Auto-sync Jira status to Slack channel", "estimated_savings_hrs": 12.5, "confidence": 0.94 } }

Encryption

AES-256 at rest, TLS 1.3 in transit

All data encrypted with customer-managed keys (BYOK) available for enterprise. mTLS for agent-to-API communication.

Access Control

RBAC + SCIM provisioning

SAML 2.0 / OIDC SSO. SCIM auto-provisioning from Okta, Azure AD, OneLogin. Granular role-based permissions per team.

Data Residency

Multi-region deployment

Choose US, EU, or APAC data residency. No data leaves your selected region. On-premise deployment available for enterprise.

Privacy

Privacy-first architecture

No screenshots, no keystroke logging, no screen recording. Workflow patterns only. Individual data isolated; managers see aggregates. Full GDPR right-to-delete via API.

Audit

Complete audit trail

Every API call, every admin action, every data access logged and immutable. Export audit logs to your SIEM (Splunk, Datadog, etc).

Network

IP allowlisting + VPN peering

Restrict API access to known IPs. VPC peering available. Agent communicates through configurable proxy. No inbound ports required.
