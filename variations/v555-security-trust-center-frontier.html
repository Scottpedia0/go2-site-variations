The Privacy-First Architecture

We designed Go2 from the ground up to ensure HR and Ops managers get the insights they need (15.2 hours saved/week avg) without compromising employee privacy.

  • Pattern Recognition, Not Content

    We track typing cadence and app switching frequency to detect fatigue. We explicitly drop the actual characters typed at the OS level.

  • No Screen Recording

    Screenshots capture sensitive data arbitrarily. Go2 relies on API integrations and app window titles, ensuring passwords and private messages are never captured.

  • Aggregated Team Insights

    Managers see trends ("The Sales team spends 40% of time in internal tools") rather than individual witch-hunts. Individual coaching is visible only to the employee and the AI.

Technical Architecture

Designed for SMBs, built to Enterprise standards. Your data is secured at every layer of the stack.

Data at Rest

All structured and unstructured data stored within Go2's infrastructure is encrypted using AES-256 block-level encryption. Keys are managed via AWS KMS.

AES-256 AWS KMS

Data in Transit

All communications between the Go2 desktop agent, web application, and our servers are encrypted via TLS 1.2 or higher, utilizing perfect forward secrecy.

TLS 1.2+ HTTPS WSS

Cloud Infrastructure

Hosted exclusively on Amazon Web Services (AWS) in isolated Virtual Private Clouds (VPCs). We utilize strict Security Groups and automated vulnerability scanning.

AWS VPC Isolation RBAC
