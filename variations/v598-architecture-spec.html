System Overview

Go2 is a distributed system consisting of lightweight desktop agents, a cloud processing pipeline, and a web-based analytics platform. The architecture prioritizes privacy-by-design, minimal data collection, and zero-trust principles.

Data Flow & Collection

Go2 collects application-level metadata only. We never capture screen content, keystrokes, file contents, or communication content. Here's exactly what flows through the system:

Data PointDescriptionRetention
app_nameActive application name (e.g., "Chrome", "Slack")90 days raw, then aggregated
window_titleWindow title (configurable: on/off/hash-only)30 days, then deleted
url_domainDomain only, never full URL (e.g., "docs.google.com")90 days raw
activity_stateActive / Idle / Away (keyboard/mouse activity flag)90 days raw
duration_msTime spent in each stateAggregated indefinitely
project_tagAI-inferred project/client association90 days raw

03

Desktop Agent Architecture

The Go2 desktop agent is a lightweight process (~15MB RAM, <0.5% CPU) that runs in the background. It uses OS-level APIs to observe application usage without requiring elevated privileges or kernel extensions.

SpecificationDetails
LanguageRust (core) + platform-specific bindings
Memory~15MB typical, 30MB peak
CPU<0.5% average
Network~2KB/min (batched every 60s)
Offline supportLocal queue, syncs when reconnected
Auto-updateDelta updates, zero-downtime, rollback support
PermissionsUser-space only. No kernel extensions, no root.

05

Security Model

Zero-trust architecture with defense-in-depth. Every component assumes breach of adjacent components.

ENCRYPTION

Data Encryption

AES-256-GCM at rest, TLS 1.3 in transit. Per-tenant encryption keys managed via AWS KMS with automatic rotation every 90 days.

AUTH

Authentication

SAML 2.0, OIDC, OAuth 2.0. MFA required for admin access. SCIM 2.0 for automated user provisioning. Session tokens rotate every 4 hours.

NETWORK

Network Security

All services in private VPC. No public endpoints except API gateway. WAF with OWASP Top 10 rules. DDoS protection via CloudFront.

COMPLIANCE

Certifications

SOC 2 Type II, GDPR compliant, HIPAA eligible (BAA available), CCPA compliant. Annual pen testing by third-party firm.

ACCESS

Access Control

RBAC with four tiers: Org Admin, Team Manager, Member, Viewer. All admin actions logged with immutable audit trail.

DATA

Data Residency

US (us-east-1, us-west-2), EU (eu-west-1, eu-central-1), APAC (ap-southeast-1). Data never leaves selected region.
